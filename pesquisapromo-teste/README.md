# PesquisaPromo Teste

Segundo site do ecossistema `PesquisaPromo`.

Objetivo:
- testar novas direcoes maiores
- prototipar outra home, outra cidade ou outra navegacao
- trabalhar sem interferir no site principal

Porta padrao:
- `localhost:3001`

Fluxo sugerido:
1. manter o site principal em `localhost:3000`
2. subir este site de testes em `localhost:3001`
3. experimentar aqui o que for praticamente uma nova versao
